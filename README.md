Tax Intern KPI Diagnostic Dashboard (MVP)

What it does

This project is a performance analytics dashboard designed to evaluate first-year tax interns during busy season using a structured, data-driven scoring system.
The system ingests task-level data (tax returns), applies predefined class weights and complexity adjustments, and evaluates performance across four core dimensions: output, efficiency, accuracy, and contribution. These components are combined into a final score that provides a transparent and defensible view of intern performance.
In addition to objective metrics, the system incorporates structured subjective inputs (flags from employees and managers) and generates diagnostic insights that highlight behavioral patterns, inconsistencies, and potential development areas. The goal is not to automate HR decisions, but to provide a consistent and interpretable decision-support layer for evaluating intern performance.
